# 기술보고서 템플릿

## 제목

MORAI SIM 기반 AI융합자율주행 시스템 설계 및 구현

## 초록

본 연구는 MORAI SIM: Drive 환경에서 ROS1 Noetic 기반 자율주행 시스템을 설계하고 구현하는 것을 목표로 한다. 대회 규정에서 허용한 GPS, IMU, 3D LiDAR, Camera 센서를 활용하여 localization, perception, planning, control 모듈을 구성하고, 시뮬레이션 기반 실험을 통해 시스템의 주행 안정성과 AI 미션 수행 가능성을 검증한다.
